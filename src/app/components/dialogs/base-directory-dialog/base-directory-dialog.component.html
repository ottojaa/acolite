Initial Application Setup

Choose a workspace directory

Base workspace directory will be the folder where all of your files created in this application go. You can choose an existing folder, create a new one, or let the application create one for you. Options are:

  • Set default directory: creates an empty folder in acolite/app
  • Choose directory: select the desired destination, or create a new one.

Change workspace on the go

Changing your workspace is easy and you can do it anytime. All of your files will still be there, and all of the workspaces will be backed up separately.

Backing up your data

By default your workspace will be backed up to the cloud if you decide to log in by one of the available authentication methods. You can control your backup preferences in the settings.
