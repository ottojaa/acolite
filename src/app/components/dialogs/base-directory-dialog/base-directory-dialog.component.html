Initial Application Setup

Choose a workspace directory

Workspace directory will be the folder where all of your files created in this application go. You can choose an existing folder, create a new one, or let the application create one for you. Options are:

  • Set default directory: creates an empty folder in documents/Acolite
  • Choose directory: select the desired destination, or create a new one.

Change workspace on the go

Changing your workspace is easy and you can do it anytime. All of your files will still be there, and all of the workspaces will be backed up separately.

Your workspace settings are automatically saved.
